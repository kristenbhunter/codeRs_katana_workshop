# CodeRs presents: Scaling up with R and Katana

Date: Thursday 4th June 2026

Have you been running analyses on your local computer that take hours or even days? This workshop is for R users who are looking to use Katana, UNSW’s cloud computing cluster, for the first time. Workshop participants should ideally have intermediate experience in R, and should already have a Katana account. The workshop will walk through the basics of getting set up running R scripts on Katana, and if time permits will discuss parallelising tasks by running multiple scripts.
